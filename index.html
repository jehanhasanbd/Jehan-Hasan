Research • 2026

Jehan Hasan

Memory framework with implicit belief state and uncertainty-gated explicit memory mechanism.

Developed a memory framework that tightly couples an implicit belief state with an uncertainty-gated explicit memory mechanism, mediated by a frozen reasoning model. Learns when to externalize structured memory, enabling bounded, interpretable inference while handling sparse evidence across arbitrarily long conversation trajectories.
